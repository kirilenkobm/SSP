## Usage

On Linux/MacOS:

```shell
git clone git@gitlab.com:kirilenkobm/ssp.git
cd ssp
make
./SSP.py test_input/3.txt 2 9
```

On windows:

```powershell
git clone git@gitlab.com:kirilenkobm/ssp.git
cd ssp
.\win_make.bat
python .\SSP.py test_input\3.txt 2 9
```

The answer for 3.txt should be [4, 5] and [3, 6]

Inputs may be generated with generate_input.py
